# AI MASTER 🎓🤖

AI MASTER is an AI-powered Intelligent Study Planning System that helps students organize their studies, generate personalized schedules, summarize learning content, assess their knowledge, and improve productivity using Artificial Intelligence.

---

## 📖 Overview

AI MASTER is designed to assist students in managing their academic tasks efficiently. The application creates personalized study plans, provides AI-generated summaries, generates assessments, and offers an AI chatbot for learning support.

---

## ✨ Features

- 🔐 User Authentication
- 📅 Intelligent Study Planner
- 📝 AI Content Summarization
- 🤖 AI Chatbot (Google Gemini)
- 📚 Daily Task Management
- 📊 Performance Dashboard
- 📈 Progress Tracking
- 🔄 Automatic Rescheduling
- ❓ AI-generated Assessments
- ☁️ Firebase Cloud Storage

## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Android Studio / VS Code
- Firebase Project
- Google Gemini API Key

### Installation

```bash
git clone https://github.com/jagadeeswararaopogiri/AI_MASTER.git

cd AI_MASTER

flutter pub get

flutter run
```
